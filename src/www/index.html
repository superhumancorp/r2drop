Frequently Asked Questions

Is R2Drop free?

Yes — R2Drop is completely free during the beta period. Download the macOS app and start uploading immediately. You only pay for Cloudflare R2 storage and egress, which is billed directly by Cloudflare (not us). A Pro plan for power users is coming soon.

Does R2Drop store my files?

No. R2Drop is a local uploader — your files are sent directly from your Mac to your Cloudflare R2 bucket. We never route, store, or cache your data on our servers. Your bucket credentials are stored exclusively in the macOS Keychain.

Can I use multiple Cloudflare R2 accounts?

Yes — R2Drop supports multiple Cloudflare accounts simultaneously. Add accounts in Settings, each with its own bucket, path prefix, and custom domain. Switch the active account from the menu bar in one click. Great for separating dev, staging, and production environments.

Does the CLI work on Linux and Windows?

Yes — the CLI is a standalone Rust binary that runs on macOS, Linux, and Windows. Install via Homebrew, the quick-install script, or build from source. It shares the same config and Keychain credentials as the macOS app when both are installed on the same machine.

Can I use R2Drop without the macOS app?

Yes — R2Drop supports multiple Cloudflare accounts. Add them in Settings, each with its own bucket, path prefix, and custom domain. Switch the active account from the menu bar.
